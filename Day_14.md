# Bubble sort
## Implement bubble sort
### Example 1:
 ```python
 # Input
 Enter numbers: 5 2 8 1 9
 # Code
 arr = [int(x) for x in input('Enter numbers: ').split()]
 bubble_sort(arr)
 print('Sorted list: ', end='')
 print(arr)
 # Output
 Enter numbers: 5 2 8 1 9
 Sorted list: [1, 2, 5, 8, 9]
 ```
